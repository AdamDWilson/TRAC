# Claude Code Instructions

BC residential tenancy template letter generator. Letters help tenants communicate with landlords following Residential Tenancy Act (RTA) requirements.

**Read `README.md`** for complete documentation on adding templates, including file structure, SurveyJS patterns, Nunjucks filters, and standard field names.

**Validate templates:** `python3 validate_templates.py`

**Run local server for browser testing:**
```bash
python3 -m http.server 8000 --directory /Users/adam/Projects/TRAC/template-letters
```
Then open http://localhost:8000 in Chrome.
